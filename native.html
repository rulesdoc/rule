所有方法都在native对象上可用
方法签名返回值说明
goLogin() void 返回到登录页，退出或切换用户
goHome() void 返回到主页
goBack() void 返回到上级页面
newPage(boolean vertical) void 打开新页面,vertical===true竖屏展示,false横屏
corsHTTP(String URL,String func) void 跨域请求,func是回调函数名
encrypt(String origin) String 加密,返回加密后的字符串
setSpeech(boolean enable) void 设置实时语音,enable===true开启,false关闭
getSpeech() boolean 返回当前实时语音启用返回true,否则返回false
doPay() String 调起在线支付,需要支持微信支付,支付宝支付,返回支付结果
doShare(String payload) void 调起分享,需要支持微信朋友圈,微信好友/群,QQ空间,QQ好友群
doDial(String callNumber) void 拨打电话
openApp(String bundleId) void 打开其它应用
renewApp() void 更新游戏到最新版本
getNetstate() String 返回当前网络状态
getLocation() String 返回当前地理位置
getDeviceID() String 返回当前设备唯一标识
setData(String key,String val) void 持久化存储数据
getData(String key) String 获取存储的数据
delData(String key) void 删除存储的数据
